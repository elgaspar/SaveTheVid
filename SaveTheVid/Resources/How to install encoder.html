How to install encoder

Step 1

Download the zip file with FFmpeg binaries from here. Select latest version, 64 or 32 bit architecture (depending on your system), static linking and click download.

Step 2

Extract the contents of the downloaded zip file.

Step 3

From the extracted files we only need ffmpeg.exe and ffprobe.exe under the bin folder.

Step 4

Copy these two files into "Resources/bin" folder in the directory you installed SaveTheVid.
Something similar to: C:\Program Files\elgaspar\SaveTheVid\Resources\bin

Step 5

Restart SaveTheVid and you are ready.
